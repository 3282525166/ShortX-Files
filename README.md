# ShortX Repo
这是ShortX发现页面的在线指令数据仓库。


## 如何贡献
往主分支提交PR，PR合并之后，索引文件会通过Github action自动更新，因此可能需要一段时间才能在App上看到最新数据。

## 文件目录

`index.json`

这是指令的索引文件，记录了所有一键指令和自动指令的基础信息以及其url

`da`

这是放一键指令的目录，每个指令对应一个文件，可以通过App分享功能直接生成文件。

`rules`

这是放自动指令的目录，每个指令对应一个文件，可以通过App分享功能直接生成文件。
